 # ⚽ Game Connect

Game Connect is a web application that helps players find and join sports games nearby.

## 💡 The Idea

This project started from a real problem.

My friends and I love playing football, but most of the time it was just 2–3 of us. Finding more players was always difficult — we didn’t know who else was playing, where they were playing, or when.

So we asked a simple question:

**What if we could know which groups are playing nearby and just join them?**

Game Connect is the solution to that.

---

## 🚀 What It Does

Game Connect allows users to:

- 🏟️ **Create a Game**  
  Add sport, location, date & time, and max players

- 🤝 **Join or Leave Games**  
  Easily participate in games nearby

- 📍 **Discover Nearby Games**  
  View games within a 25 km radius

- 👥 **Track Player Count**  
  See how many players have joined vs available spots

- 📱 **Responsive UI**  
  Works across devices with a clean interface
